This repository contains the documentation, code, and tutorials for the Introduction to Robotics project, focusing on the following core applications:

- Electrical: It contains the IO Diagram.

- HMI (Human-Machine Interface): Contains the files (code, configuration) for the control panel developed for real-time interaction and monitoring of the robot.

- RC+5.0 Program: 

- Task #1 - Pick and place: Includes the code and documentation for the fundamental task of picking up an object and placing it in a specific location.

- Task #2 - Stacking: As the Task 1, this one includes the code and documentation for the fundamental task of stacking the tokens in a specific location.

- Task #3 - This file contains the CAD files and all the documentation (including images and videos)  of this task.

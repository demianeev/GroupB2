This repository contains the documentation, code, and tutorials for the Introduction to Robotics project, focusing on the following core applications:

- HMI (Human-Machine Interface): Contains the files (code, configuration) for the control panel developed for real-time interaction and monitoring of the robot.

- Task #1 - Pick and place: Includes the code, strategies, and documentation for the fundamental task of picking up an object and placing it in a specific location.

- Task #2 - Stacking: Contains the code and CAD files.
